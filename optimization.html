🎯 กระบวนการ Train Model

การ train model เป็น iterative process ในแต่ละ iteration:

  1. Model ทำนายผลลัพธ์
  2. คำนวณ loss (ความผิดพลาด)
  3. คำนวณ gradients ย้อนกลับ
  4. ปรับ parameters ด้วย optimization

🔧 Hyperparameters

Hyperparameters คือ parameters ที่เรากำหนดเองเพื่อควบคุมการ train

📅 Epochs

5

จำนวนรอบที่ loop ผ่าน dataset ทั้งหมด

📦 Batch Size

จำนวน samples ก่อนอัพเดท parameters

📈 Learning Rate

ขนาดก้าวในการปรับ parameters

learning_rate = 1e-3 batch_size = 64 epochs = 5

📝 Loss Function

Loss function วัดความผิดพลาดระหว่าง prediction กับ target

Loss Functions ที่ใช้บ่อย

Loss Function ใช้กับ
nn.MSELoss Regression (ค่าต่อเนื่อง)
nn.NLLLoss Classification (Negative Log Likelihood)
nn.CrossEntropyLoss Classification (รวม LogSoftmax + NLLLoss)
# สร้าง loss function loss_fn = nn.CrossEntropyLoss()

🚀 Optimizer

Optimizer ปรับ parameters ตาม gradients

optimizer = torch.optim.SGD(model.parameters(), lr=learning_rate)

Optimizers ที่นิยม

  • SGD - Stochastic Gradient Descent
  • Adam - Adaptive Moment Estimation (นิยมมาก)
  • RMSprop - Root Mean Square Propagation

🔄 Training Loop

แต่ละ epoch ประกอบด้วย:

1
Forward Pass - ทำนายผลลัพธ์
pred = model(X) loss = loss_fn(pred, y)
2
Zero Grad - ล้าง gradients เก่า
optimizer.zero_grad()
3
Backward Pass - คำนวณ gradients ใหม่
loss.backward()
4
Update Parameters - ปรับ parameters
optimizer.step()
