📥 ToTensor - แปลงเป็น Tensor

แปลง PIL Image หรือ NumPy array เป็น PyTorch Tensor และ scale ค่า pixel เป็น [0, 1]

📷 ก่อน (PIL Image)

ค่า 0-255

🔢 หลัง (Tensor)

ค่า 0.0-1.0

Code:

from torchvision.transforms import ToTensor

transform = ToTensor()

# แปลง PIL Image เป็น Tensor
img_tensor = transform(img_pil)

🔧 Transforms ที่ใช้บ่อย

Transform หน้าที่ ตัวอย่าง
ToTensor() แปลงเป็น Tensor [0,1] PIL → Tensor
Normalize() Normalize ค่า mean, std
Resize() ปรับขนาด (H, W) → (size)
CenterCrop() ตัดกลาง ตัดตรงกลาง
RandomHorizontalFlip() พลิกซ้าย-ขวา Data Augmentation
RandomRotation() หมุนสุ่ม Data Augmentation
Lambda() Function กำหนดเอง เขียนเอง

🔗 Compose - รวมหลาย Transforms

ใช้ Compose รวม transforms หลายตัวเข้าด้วยกัน

from torchvision.transforms import Compose, Resize, ToTensor, Normalize

# รวม transforms หลายตัว
transform = Compose([
    Resize((224, 224)),    # 1. ปรับขนาด
    ToTensor(),             # 2. แปลงเป็น Tensor
    Normalize(              # 3. Normalize
        mean=[0.485, 0.456, 0.406],
        std=[0.229, 0.224, 0.225]
    )
])

📝 ตัวอย่างเต็ม

import torch
from torchvision import datasets
from torchvision.transforms import ToTensor, Lambda, Compose

# กำหนด transforms
transform = Compose([
    ToTensor(),
])

target_transform = Lambda(lambda y: torch.zeros(
    10, dtype=torch.float
).scatter_(0, torch.tensor(y), value=1))

# ใช้กับ Dataset
training_data = datasets.FashionMNIST(
    root="data",
    train=True,
    download=True,
    transform=transform,
    target_transform=target_transform
)
